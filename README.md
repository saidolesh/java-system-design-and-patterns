# java-system-design-and-patterns
Comprehensive Implementation of Design Patterns, Low Level Design, and System Design concepts in Java.

1. Singleton Pattern

Singleton is: A creational Design Pattern that ensures only one instance of class exist & provide global access to it..

Singleton is mainly used in:

(i) Logging Systems
(ii) Database Connection Pools
(iii) Configuration managers
(iv) Caching Systems
